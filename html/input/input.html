Input

Input Type

  • checkbox
  • color
  • radio
  • password
  • file
  • range
  • text
  • select
  • button

text

  • name 속성은 폼(form)이 제출된 후 서버에서 폼 데이터(form data)를 참조하기 위해 사용되거나, 자바스크립트에서 요소를 참조하기 위해 사용됨.
  • size로 가로크기를 조절가능 width속성으로도 가능
  • value로 안에 적거나 읽어올 수 있음
  • readonly로 value를 읽기전용속성으로 가능
  • placeholder로 안에 value가 없을때 텍스트를 넣을 수 있음
  • maxlength, minlength로 최대글자수 최소글자수 제한가능
  • required를 적으면 폼 데이터가 서버로 제출되기 전 반드시 채워져 있어야 하는 입력필드를 명시한다.
    required는 checkbox, date, email, file, number, password, radio, text 등 에서도 사용가능하며 bolean 속성이다.

password

  • text와 비슷하나 value가 눈에 보이지않게 해줌 -> 비밀번호 태그로 사용

radio

  • 여러 개의 항목 중에서 하나만 선택하도록 하는 동그란버튼을 라디오 버튼이라고 함
  • name 속성을 이용해 name값이 같은 것 중에서 하나를 선택할 수 있게함
  • value를 설정해 놓으면 선택했을 때 전달될 값이 된다.
  • checked 선택된 상태로 만듦
  • label을 이용해 텍스트를 눌러도 라디오가 선택되게 할 수 있다. label에서 for속성을 radio버튼의 id명에 맞춰놔야함 name은 안됨

checkbox

  • 체크박스는 클릭하면 선택되고, 다시 클릭하면 해제된다. radio와는 다르게 다중선택이 가능
  • name 속성은 전달할 값의 이름이다.
  • value 속성은 전달될 값이다.
  • checked로 선택된 상태로 만들 수 있다.
  • radio와 같게 label은 id명에 맞춰 셋팅하면 클릭할 수 있다.
    같은 id로 해놓으면 맨위에거에만 적용된다.

select

  • name 속성은 전달할 값의 이름
  • autofocus 속성으로 페이지가 로드되었을 떄 해당 목록으로 바로 포커싱 됨
  • disable 속성을 사용하면 화면에는 select박스가 보이지만 선택을 할 수 없게됨
  • multiple 속성을 사용하면 여러 값을 선택할 수 있도록 한다.
  • size 속성은 목록을 펼치기 전에 보여질 목록의 개수를 지정할 수 있다.
  • select안에는 option태그를 사용할 수 있고 option태그에 value를 지정해놓으면 됨
  • option태그에서 selected를 사용해 먼저 선택되어있게 할 수 있다. radio나 checkbox의 checked속성 느낌
  • optgroup 태그를 사용해 카테고리 처럼 만들기도 가능하다.
  • select박스 자체는 css로 디자인이 가능하다.(label을 이용해서 디자인도 가능)
    하지만 option은 따로 디자인이 되지않아 아예 select box를 커스텀해서 사용하는 것도 좋다.

file

  • 첨부파일을 업로드 하는 경우에 사용
  • accept 속성을 사용해 입력받을 수 있는 파일의 유형을 지정할 수 있다. 지정하지 않을시 모든 유형의 파일을 받을 수 있음
    확장자명을 사용(.txt, .pdf) , MIME타입 사용(text/plain, image/speg, ...)
  • multiple 속성으로 파일 다중선택가능 단축키 shift + ctrl + 마우스클릭
  • 주로 커스텀을 해서 사용을 많이하고
    $('.file').change(function(){
    console.log($(this)[0].files[0].name)
    console.log($(this).val().split('/').pop().split('\\').pop())
    })

    이런식으로 파일명을 꺼내올 수 있다.

range


  • 슬라이드 바를 조정하여 범위 내의 숫자를 선택할 수 있는 필드릅 정의함
  • 기본 범위는 0부터 100이며 그 범위를 설정가능하다
  • max속성을 이용해서 최대값 min속성을 이용해 최소값을 설정가능함
  • step속성을 이용해 숫자들 사의 간격을 지정
  • value 속성을 이용해 초기값을 명시할 수 있음
  • 밸류넣을곳.textContent = input.value;
    range인풋요소.addEventListener("input", (event) => {
    밸류넣을곳.textContent = event.target.value;
    });
    로 value값을 가져올 수 있다
  • datalist 태그와 안에 option태그를 사용해 지점을 정할 수 있다.

color

  • 색상을 지정할 수 있다
  • 입력 기본값은 #000000 검정색이며 반드시 해시문자(#)로 시작하는 7자리의 16진수의 값이 와야한다.
